# Saree & Co. Financial & Manufacturing Analytics Dashboard

## Project Overview

This project demonstrates an end-to-end Financial Analytics solution built using Tally Prime, Microsoft Excel, and Power BI.

The objective was to transform raw accounting transactions into meaningful business insights through interactive dashboards and KPI monitoring.

---

## Tools Used

* Tally Prime
* Microsoft Excel
* Power BI
* DAX
* Data Modeling

---

## Dashboard Pages

* Total Sales

* Net Profit
* Profit Margin

### Sales Analysis

* Customer-wise Sales

* Supplier Analysis

* GST Collection

### Cash Flow Dashboard

* Receipts vs Payments
